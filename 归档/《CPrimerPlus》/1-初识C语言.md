# 初识C语言

## C语言的优缺点

### 优点

- 设计特性
- 高效性
- 可移植性
- 强大而灵活
- 面向程序员

### 缺点

- 指针和大量的运算符给予了程序员更多的自由，容易引起混乱

## 使用C语言的7个步骤

1. 定义程序的目标
2. 设计程序
3. 编写代码
4. 编译
5. 运行代码
6. 测试和调试程序
7. 维护和修改程序
