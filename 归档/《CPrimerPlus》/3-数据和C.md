# 数据和C

## 变量和常量

- 常量：在程序使用之前已经预先设定好数据类型的数据，在整个程序运行过程中没有变化
- 变量：在程序运行期间可能会被改变或者赋值的数据

## C语言基本数据类型（P60 总结）

- 整数类型
  - int
  - char
  - _Bool
- 浮点数类型
  - float
    - 至少能够表示6位有效数字
  - double
    - 至少能表示10位，但是实现时至少13位
    - 浮点型常量默认double精度
  - long double
  - 特殊浮点值
    - 上溢：inf
    - 下溢：精度损失
    - 非数：NAN
- sizeof %zd size_t

## 转义序列

- 常用转义字符
  - \b	退格，光标向后退一格
  - \n     换行，光标置下一行行首
  - \r      回车，光标置当前行行首

## 缓冲区

printf（）语句把输出发送到叫做缓冲区的中间存储区域，当

- 缓冲区满
- 遇到换行符
- 或需要输入

的时候，刷新缓冲区，缓冲区将数据发送到目标。
