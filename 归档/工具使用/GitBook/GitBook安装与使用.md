# 安装与使用

## [参考链接](https://blog.csdn.net/lu_embedded/article/details/81100704)

`安装环境：ArchLinux`

## 安装-npm

```shell
npm install -g gitbook-cli
```

## 使用

1. 新建文件夹（作为书本载体目录）

2. 输入命令

   ```shell
   gitbook init
   ```

3. 本地服务器预览

   ```shell
   gitbook serve
   ```

4. 生成静态网页文件

   ```shell
   gitbook build
   ```
